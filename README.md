# SimpleK9 - LSPDFR Plugin

A basic K9 plugin for LSPDFR.

## Features
- Spawn German Shepherd
- Follow
- Sit
- Bite marked suspect
- Enter/Exit vehicle
- Retire mode

## Controls
- Num* → Open Menu (Numberpad StarKey)
- U → Mark Suspect

## Installation
Drop the .dll into:
Grand Theft Auto V/Plugins/LSPDFR/

Built for RagePluginHook & LSPDFR
